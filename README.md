# 🎯 Aim Trainer Pro

A fast-paced and interactive **reaction & aiming web application** designed to improve user reflexes and accuracy.  
Built with a responsive UI and multiple gameplay modes, this project demonstrates real-time interaction, animation handling, and event-driven programming in JavaScript.

---

## 🚀 Live Demo
👉 https://time-game-drab.vercel.app


## 🎮 Features

- 🎯 Classic Mode – Smooth bouncing target
- ⚡ Reaction Mode – Random target appearance for reflex testing
- 🎯 Precision Mode – Smaller target for accuracy challenge
- 💥 Chaos Mode – Multiple moving targets simultaneously
- 📱 Fully Responsive (Mobile + Desktop)
- 👆 Touch + Click Input Support
- 💥 Hit Animations (Visual Feedback)
- 🏆 Game Over Popup Screen
- 🎚 Difficulty Levels (Easy / Medium / Hard)
- 🎮 Fullscreen Mode for immersive gameplay

## ⚙️ How It Works

- Uses `requestAnimationFrame` for smooth target movement  
- Event listeners handle both **click and touch interactions**  
- Dynamic difficulty adjustment based on selected level  
- Real-time score tracking and timer management
